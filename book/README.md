## Running consistency watch
Things that must stay true across every chapter:
- **Three orientations per gradient** — two poles + a balanced middle (Spatial: Concrete/Balanced/Abstract · Temporal: Past/Present/Future · Reference: Self/Balanced/World). Present is the *seam orientation,* not a deletion. **⚠️ Ch.9 frames configuration as "weight toward A or B" and never names the balanced middle — must be reconciled or a reader infers 2³=8 types, not 3³=27.**
- **Parser ≠ personality** — a Parser is the *reception configuration beneath* the output; personality systems measure the composite output. (Stated cleanly in Ch.9 — make every other chapter honor it.)
- **Blocks are convergence hubs, not gradients** — a block is never "a position on an axis." (Ch.9: "Brain regions are not the gradients themselves… convergence sites.")
- **Gradients = reception/direction; the conveyor (rostral-caudal) = delivery.** Keep them separate. **⚠️ rostral-caudal ≡ anterior-posterior — the same physical axis Ch.9 assigns to temporal reception. When the conveyor chapter lands, it must explain how one axis carries both.**
- **Fixed architecture + learned incorporation** — positions don't move; bridges get built. (Ch.9: "Configuration Is Not Capacity.")
- **Anatomical claims:** confident for the lay/buyer audience; hedge to "maps onto known cortical gradients" if a passage is aimed at scientists/clinicians. **Weakest leg = reference/medial-lateral (no oscillatory citation; egocentric-allocentric isn't classically a medial-lateral gradient) — hedge it like the Margulies caveat.**
