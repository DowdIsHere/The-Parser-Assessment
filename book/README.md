## Running consistency watch
Things that must stay true across every chapter:
- **Three orientations per gradient** — two poles + a balanced middle (Spatial: Concrete/Balanced/Abstract · Temporal: Past/Present/Future · Reference: Self/Balanced/World). Present is the *seam orientation,* not a deletion.
- **Blocks are convergence hubs, not gradients** — a block is never "a position on an axis."
- **Gradients = reception/direction; the conveyor (rostral-caudal) = delivery.** Keep them separate.
- **Fixed architecture + learned incorporation** — positions don't move; bridges get built.
- **Anatomical claims:** confident for the lay/buyer audience; hedge to "maps onto known cortical gradients" if a passage is aimed at scientists/clinicians.
