## TEMPLATE USAGE INSTRUCTIONS

This template generates a complete Children Report from a child's Parser Profile result. The report has **two parts**:
1. **For the Child** — Written at their reading level, celebrates their architecture
2. **For the Parents** — Written in two versions (one for each parent's Parser), includes the behavioral reality

**Required Inputs:**
- Child: Name, Age, Profile Name, Coordinates, Position Values
- Parent A: Name, Profile Name, Coordinates (for tailored parent guidance)
- Parent B: Name, Profile Name, Coordinates (for tailored parent guidance)

**Output:** 3,000–5,000 word parent-facing report with child-facing section

Because you each take in information differently, there are two versions of the parent guidance—same core insights, delivered in the register each of you hears best.
